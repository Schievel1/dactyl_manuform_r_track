# Soundboard keymap

This keymap binds all keys to function keys 13-22 for something like a
soundboard (ie:[Resanance](https://www.reddit.com/r/discordapp/comments/44ldc1/soundboard_for_discord/))
```
//         ┌────────┬────────┬────────┐

              F13   ,  F14   ,   F15  ,

//    ├────────┼────────┼────────┼────────┼

          F16  ,   F17  ,   F18  ,  F19   ,

//    ├────────┼────────┼────────┼────────┼

                F20 ,   F21   ,  F22

//         └────────┴────────┴────────┘
```
