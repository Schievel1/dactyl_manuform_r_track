# Simple numberpad keymap

This keymap is to test the soldering work of workshop participants. It can also be used as a simple base for a macropad keymap.
```
//         ┌────────┬────────┬────────┐

                1   ,   2   ,    3    ,

//    ├────────┼────────┼────────┼────────┼

          4    ,   5    ,    6   ,    7   ,

//    ├────────┼────────┼────────┼────────┼

                8   ,    9   ,    0

//         └────────┴────────┴────────┘
```
